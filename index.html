How plazd can help campus placements?

Simplified application collection

Most of the placement office notifies students about companies(visiting campus) opening through email and facebook group. This is bizzare. plazd makes it easy to host form and collect data easily.

Reduces Time to organise..

Using different platform(fb, email, google sheet, form etc) = Huge time consumption. plazd brings all these utility into one platform, thereby reduce time to operate.

Gold mine of data insights*

Placement process do generate a lot of data. But this data is not used properly. plazd work to aggregate this data with analytics to drive placement season of college more efficiently.

Reduces Bias

Different Students == Different Resume.
This difference unknowningly creates bias in sorting procedure for next round. plazd generalises the profile of students helping each student to show thier best capability. This also help placement office to verify student's profile more effectively.

Hosted using Node.JS
